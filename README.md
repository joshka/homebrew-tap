# Joshka Tap

## How do I install these formulae?

`brew install joshka/tap/<formula>`

Or `brew tap joshka/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "joshka/tap"
brew "<formula>"
```
